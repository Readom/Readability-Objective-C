Readability-Objective-C

Metrics to determine readability and comprehension difficulty for contemporary English text.

  • Example app and unit tests are included.
  • The Smalltalk version is here.
  • The Swift version is here.
Formulas Included
  • Automated Readability Index
  • Flesch-Kincaid Grade Level
  • Flesch Reading Ease
  • Gunning Fog Index
  • SMOG

Dependencies

  • objC-syllable-counter with RegexKitLite (included) and libicucore

Usage

See included example application.
