Readability-iOS

iOS tools to test readability and comprehension difficulty for contemporary English text.

  • The Swift version is here.
  • Unit tests are included.
  • The Readability Test Tool has been used during development.
Formulas Included
  • Automated Readability Index
  • Flesch-Kincaid Grade Level
  • Flesch Reading Ease
  • Gunning Fog Index
  • SMOG

Dependencies

  • objC-syllable-counter with RegexKitLite (included) and libicucore

Usage

See included example application.
